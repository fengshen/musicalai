# Errors                                                                                                                                                        
  
The musicalai API uses the following error codes:
 
 
Error Code | Meaning
---------- | -------
200| OK
201| Done
400| Bad Request
401| Unauthorized
404| Not Found
405| Method Not Allowed
409| Conflict
415| Unsupported Media Type
422| Unprocessable
429| Too Many Requests
